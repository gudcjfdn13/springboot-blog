# springboot-blog

# 기술스택
- JDK 1.8
- MyBatis
- JDBC
- Spring Boot 2
- MySQL

# 개발 툴
- Maven
- sts4.7.2
- Visual Studio Code
- Whale
- Git, GitHUB

# 구현기능 리스트
- 게시물 리스팅/페이징, 작성, 수정, 삭제
- 로그인, 회원가입, 로그아웃
- 회원정보 수정
- 검색기능
- 댓글 리스팅, 작성, 삭제
- 댓글관련 페이지 이동시 게시물 검색어, 페이지 유지
- 게시판 추가(notice, free)

# 구현예정 리스트
- 회원가입시 이메일 인증 추가, 이메일 중복체크
- 아이디, 비밀번호 찾기
- 회원가입, 비밀번호 찾기 이메일 주소로 안내 메일 발송
- 비밀번호 암호화
